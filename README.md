# Clean Screenshot

Extension Chrome pour nettoyer une page web et prendre une capture d’écran avec un son d’appareil photo.

## Fonctionnalités
- Nettoie la page avant la capture
- Prend une capture d’écran de l’onglet actif
- Joue un son d’appareil photo lors de la capture

## Installation
1. Téléchargez le code ou clonez le dépôt.
2. Rendez-vous sur `chrome://extensions` dans Chrome.
3. Activez le mode développeur.
4. Cliquez sur « Charger l’extension non empaquetée » et sélectionnez le dossier du projet.

## Utilisation
Cliquez sur l’icône de l’extension pour nettoyer la page et prendre une capture d’écran.
